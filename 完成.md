## 🚀 快速开始（推荐：Ngrok）

### 步骤 1：注册 Ngrok（5 分钟）

1. 访问：https://dashboard.ngrok.com/signup
2. 使用邮箱注册（免费，无需信用卡）
3. 验证邮箱
4. 登录后复制 authtoken

### 步骤 2：配置（1 分钟）

```bash
ngrok config add-authtoken YOUR_TOKEN_HERE
```

将 `YOUR_TOKEN_HERE` 替换为您复制的 authtoken

### 步骤 3：启动隧道（1 分钟）

```bash
cd /home/ming/.openclaw/workspace-yuxin/future-you
./ngrok-start.sh
```

### 步骤 4：获取公网 URL

```bash
./ngrok-urls.sh
```

您会看到类似这样的输出：

```
🌐 Frontend URL:
https://abcd-1234.ngrok-free.app

📧 Backend URL:
https://wxyz-5678.ngrok-free.app
```

### 步骤 5：测试访问

在浏览器中访问显示的前端 URL，应该能看到 Future You 的界面！

## 💡 重要提示

### URL 变化说明
- 免费隧道每次重启 URL 可能会变化
- 每次启动后运行 `./ngrok-urls.sh` 获取最新 URL
- 如需固定 URL，需要升级到付费版

### 备选方案
如果 ngrok 不适用，可以尝试：
```bash
# 使用 localtunnel（可能不稳定）
./tunnel-start.sh
```

### 故障排除
- **authentication failed**：需要先配置 authtoken
- **端口已被占用**：运行 `./ngrok-stop.sh` 停止旧隧道
- **连接不稳定**：尝试使用 ngrok 代替 localtunnel
