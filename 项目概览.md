## API接口文档

### 认证相关（公开接口）

| 方法 | 路径 | 描述 | 请求体 |
|------|------|------|--------|
| POST | `/api/auth/register` | 用户注册 | username, password, confirmPassword, phone?, email? |
| POST | `/api/auth/login` | 用户登录 | username, password |
| POST | `/api/auth/check-username` | 检查用户名可用性 | username |

### 用户相关（需认证）

所有接口需要请求头：`Authorization: Bearer <token>`

| 方法 | 路径 | 描述 | 请求体 |
|------|------|------|--------|
| GET | `/api/user/info` | 获取当前用户信息 | - |
| PUT | `/api/user/info` | 更新用户信息 | phone?, email? |
| DELETE | `/api/user/account` | 删除账号 | - |

### 响应格式

#### 成功响应
```json
{
  "message": "操作成功",
  "token": "jwt-token-string",
  "user": {
    "_id": "user-id",
    "username": "username",
    "phone": "phone-number",
    "email": "email-address",
    "createdAt": "2024-03-17T00:00:00.000Z"
  }
}
```

#### 错误响应
```json
{
  "message": "错误描述",
  "errors": [
    {
      "msg": "具体错误信息",
      "param": "参数名",
      "location": "body"
    }
  ]
}
```

## 快速开始

### 前置条件
- Node.js >= 18.0.0
- MongoDB >= 4.4

### 启动步骤

1. **启动 MongoDB**
   ```bash
   # 使用 systemctl
   sudo systemctl start mongod

   # 或使用 Docker
   docker run -d --name future-you-mongodb -p 27017:27017 mongo:6.0
   ```

2. **启动后端**
   ```bash
   cd server
   npm run dev
   ```

3. **启动前端**
   ```bash
   cd client
   npm run dev
   ```

4. **访问应用**
   ```
   打开浏览器访问: http://localhost:5173
   ```

详细说明请参考 [启动指南.md](./启动指南.md)
