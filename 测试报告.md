## ⚠️ 注意事项

1. **数据备份**
   - JSON 文件是数据存储的核心
   - 建议定期备份 `server/data/` 目录

2. **并发限制**
   - JSON 文件存储不支持高并发场景
   - 如果需要处理大量并发请求，建议升级到真正的数据库

3. **生产环境安全**
   - 生产环境应使用更强的 JWT_SECRET
   - 考虑使用 HTTPS
   - 实现更严格的 CORS 配置

4. **外网访问**
   - 确保路由器配置了端口转发
   - 建议使用防火墙限制不必要的访问
   - 生产环境应使用反向代理（如 Nginx）
