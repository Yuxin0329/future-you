# 静态资源目录

此目录用于存放静态资源文件，例如：

- 图片（images）
- 字体（fonts）
- 其他静态文件

## 当前资源

- `styles/main.scss` - 全局样式文件

## 可添加的资源

### Logo/图标
- `logo.svg` - 应用Logo
- `favicon.ico` - 浏览器图标

### 图片
- `images/banner.jpg` - 首页Banner
- `images/hero.png` - 英雄区域插图
- `images/empty.png` - 空状态图

### 字体
- `fonts/` - 自定义字体文件（如果需要）

## 注意事项

1. 图片资源建议使用 WebP 格式以优化加载速度
2. SVG 图标可以直接内联到组件中使用
3. 大型图片建议使用 CDN 或对象存储
