```bash

nohup npm run dev > /tmp/future-you-frontend.log 2>&1 &

# 启动 ngrok 隧道
nohup ngrok http 5173 --log=stdout --log-level=info > /tmp/ngrok-tunnel.log 2>&1 &
```

## 🛠️ 故障排除

### 问题1：无法访问前端
**解决方法：**
1. 检查 ngrok 隧道是否运行：
   ```bash
   cat /tmp/ngrok-tunnel.log
   ```

2. 检查前端服务是否运行：
   ```bash
   ps aux | grep vite | grep -v grep
   ```

3. 如果没有运行，重启：
   ```bash
   cd /home/ming/.openclaw/workspace-yuxin/future-you/client
   nohup npm run dev > /tmp/future-you-frontend.log 2>&1 &
   ```

### 问题2：无法注册/登录
**解决方法：**
1. 检查后端服务是否运行：
   ```bash
   ps aux | grep "node.*app.js" | grep -v grep
   ```

2. 测试 API：
   ```bash
   curl https://unlucky-erik-emboly.ngrok-free.dev/api/health
   ```

3. 如果返回错误，重启后端：
   ```bash
   cd /home/ming/.openclaw/workspace-yuxin/future-you/server
   npm run dev
   ```

### 问题3：API 请求失败
**解决方法：**
1. 清除浏览器缓存并刷新页面
2. 检查浏览器控制台是否有错误信息
3. 如果有 CORS 错误，检查 vite.config.js 中的代理配置
