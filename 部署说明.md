## 环境准备

### 系统要求

- **操作系统**: Linux (推荐 Ubuntu 20.04+), macOS, 或 Windows
- **Node.js**: >= 18.0.0
- **MongoDB**: >= 4.4
- **内存**: 至少 2GB RAM
- **存储**: 至少 10GB 可用空间

### 安装 Node.js

#### Linux (Ubuntu/Debian)
```bash
# 使用 NodeSource 仓库安装
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 验证安装
node -v
npm -v
```

#### macOS
```bash
# 使用 Homebrew 安装
brew install node

# 验证安装
node -v
npm -v
```

#### Windows
从 [Node.js 官网](https://nodejs.org/) 下载并安装 LTS 版本

### 安装 MongoDB

#### Linux (Ubuntu/Debian)
```bash
# 导入 MongoDB 公钥
wget -qO - https://www.mongodb.org/static/pgp/server-6.0.asc | sudo apt-key add -

# 添加 MongoDB 仓库
echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/6.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-6.0.list

# 更新包列表
sudo apt-get update

# 安装 MongoDB
sudo apt-get install -y mongodb-org

# 启动 MongoDB
sudo systemctl start mongod
sudo systemctl enable mongod
```

#### macOS
```bash
# 使用 Homebrew 安装
brew tap mongodb/brew
brew install mongodb-community

# 启动 MongoDB
brew services start mongodb-community
```

#### Windows
从 [MongoDB 官网](https://www.mongodb.com/try/download/community) 下载并安装

## 本地部署

### 1. 克隆项目（如果从 Git 仓库）

```bash
git clone <repository-url>
cd future-you
```

### 2. 安装依赖

```bash
# 安装前端依赖
cd client
npm install

# 安装后端依赖
cd ../server
npm install
```

### 3. 配置环境变量

编辑 `server/.env` 文件：

```env
# 服务器配置
PORT=3000
NODE_ENV=production

# MongoDB配置
MONGODB_URI=mongodb://localhost:27017/future-you

# JWT密钥（请使用更复杂的密钥）
JWT_SECRET=your-very-secure-secret-key-change-this-in-production
JWT_EXPIRES_IN=7d

# CORS配置
CORS_ORIGIN=http://localhost:5173
```

### 4. 构建前端

```bash
cd client
npm run build
```

### 5. 启动后端服务器

```bash
cd server
npm start
```

### 6. 配置 Nginx（可选但推荐）

安装 Nginx：
```bash
sudo apt-get install nginx  # Linux
brew install nginx         # macOS
```

配置 Nginx 反向代理：

编辑 `/etc/nginx/sites-available/future-you`：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    # 前端静态文件
    location / {
        root /path/to/future-you/client/dist;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API
    location /api {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：
```bash
sudo ln -s /etc/nginx/sites-available/future-you /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

## 云服务器部署

### 准备工作

1. 购买云服务器（推荐：阿里云、腾讯云、AWS、DigitalOcean）
2. 选择配置：2核4GB 或更高
3. 安装 Ubuntu 20.04 或 22.04
4. 配置安全组，开放 80、443、22 端口

### 部署步骤

#### 1. 连接到服务器

```bash
ssh root@your-server-ip
```

#### 2. 安装必要软件

```bash
# 更新系统
sudo apt-get update && sudo apt-get upgrade -y

# 安装 Node.js
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# 安装 MongoDB
wget -qO - https://www.mongodb.org/static/pgp/server-6.0.asc | sudo apt-key add -
echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/6.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-6.0.list
sudo apt-get update
sudo apt-get install -y mongodb-org

# 安装 Nginx
sudo apt-get install -y nginx

# 安装 Git
sudo apt-get install -y git

# 安装 PM2（进程管理器）
sudo npm install -g pm2
```

#### 3. 启动 MongoDB

```bash
sudo systemctl start mongod
sudo systemctl enable mongod
```

#### 4. 克隆项目

```bash
cd /var/www
git clone <repository-url>
cd future-you
```

#### 5. 安装依赖并构建

```bash
# 安装前端依赖
cd client
npm install
npm run build

# 安装后端依赖
cd ../server
npm install
```

#### 6. 配置环境变量

编辑 `server/.env`：

```env
PORT=3000
NODE_ENV=production
MONGODB_URI=mongodb://localhost:27017/future-you
JWT_SECRET=your-production-secret-key-must-be-very-long-and-random
JWT_EXPIRES_IN=7d
CORS_ORIGIN=https://your-domain.com
```

#### 7. 使用 PM2 启动应用

```bash
cd /var/www/future-you/server

# 启动应用
pm2 start app.js --name "future-you"

# 设置开机自启
pm2 startup
pm2 save

# 查看状态
pm2 status

# 查看日志
pm2 logs future-you
```

#### 8. 配置 Nginx

创建 Nginx 配置文件：

```bash
sudo nano /etc/nginx/sites-available/future-you
```

添加以下内容：

```nginx
server {
    listen 80;
    server_name your-domain.com;

    # 前端静态文件
    location / {
        root /var/www/future-you/client/dist;
        try_files $uri $uri/ /index.html;
    }

    # 后端 API
    location /api {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：

```bash
sudo ln -s /etc/nginx/sites-available/future-you /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx
```

## 生产环境配置

### HTTPS 配置（推荐）

使用 Let's Encrypt 免费证书：

```bash
# 安装 Certbot
sudo apt-get install certbot python3-certbot-nginx

# 获取证书
sudo certbot --nginx -d your-domain.com

# 自动续期
sudo certbot renew --dry-run
```

### 安全加固

1. **配置防火墙**

```bash
# 安装 UFW
sudo apt-get install ufw

# 允许 SSH
sudo ufw allow 22/tcp

# 允许 HTTP 和 HTTPS
sudo ufw allow 80/tcp
sudo ufw allow 443/tcp

# 启用防火墙
sudo ufw enable
```

2. **配置 MongoDB 认证**

编辑 `/etc/mongod.conf`：

```yaml
security:
  authorization: enabled
```

重启 MongoDB：
```bash
sudo systemctl restart mongod
```

3. **使用强密码**

确保所有环境变量中的密钥和密码都是强密码。

### 性能优化

1. **启用 Gzip 压缩**

在 Nginx 配置中添加：

```nginx
gzip on;
gzip_vary on;
gzip_min_length 1024;
gzip_types text/plain text/css text/xml text/javascript application/x-javascript application/xml+rss application/json;
```

2. **配置缓存**

添加静态资源缓存：

```nginx
location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
    expires 1y;
    add_header Cache-Control "public, immutable";
}
```

3. **MongoDB 性能调优**

- 添加适当的索引
- 使用连接池
- 定期备份数据

### 备份策略

#### MongoDB 备份

```bash
# 创建备份脚本
nano /home/backup-mongodb.sh
```

```bash
#!/bin/bash
DATE=$(date +%Y%m%d_%H%M%S)
BACKUP_DIR="/home/backups"
MONGODB_URI="mongodb://localhost:27017"

mkdir -p $BACKUP_DIR

mongodump --uri="$MONGODB_URI" --out="$BACKUP_DIR/future-you-$DATE"
tar -czf "$BACKUP_DIR/future-you-$DATE.tar.gz" "$BACKUP_DIR/future-you-$DATE"
rm -rf "$BACKUP_DIR/future-you-$DATE"

# 保留最近7天的备份
find $BACKUP_DIR -name "future-you-*.tar.gz" -mtime +7 -delete
```

```bash
chmod +x /home/backup-mongodb.sh

# 添加到 crontab（每天凌晨2点备份）
crontab -e
```

添加：
```
0 2 * * * /home/backup-mongodb.sh
```

## 常见问题

### Q: MongoDB 连接失败

**A:** 检查 MongoDB 是否运行：
```bash
sudo systemctl status mongod
```

如果不是 `active` 状态，启动它：
```bash
sudo systemctl start mongod
```

### Q: 端口被占用

**A:** 检查端口占用：
```bash
sudo lsof -i :3000
```

杀死占用端口的进程或修改 `.env` 中的 `PORT` 配置。

### Q: PM2 进程崩溃

**A:** 查看日志：
```bash
pm2 logs future-you
```

重启进程：
```bash
pm2 restart future-you
```

### Q: Nginx 502 错误

**A:** 检查后端服务是否运行：
```bash
pm2 status
```

如果后端服务未运行，启动它：
```bash
cd /var/www/future-you/server
pm2 start app.js --name "future-you"
```
