```bash

nohup npx localtunnel --port 3000 --subdomain future-you-backend > /tmp/lt-backend.log 2>&1 &
```

## 🛠️ 故障排除

### 问题1：无法访问前端
**解决方法：**
1. 检查前端隧道是否运行：
   ```bash
   cat /tmp/lt-frontend.log
   ```

2. 如果没有运行，重启：
   ```bash
   nohup npx localtunnel --port 5173 --subdomain future-you-frontend > /tmp/lt-frontend.log 2>&1 &
   ```

### 问题2：无法注册/登录
**解决方法：**
1. 检查后端隧道是否运行：
   ```bash
   cat /tmp/lt-backend.log
   ```

2. 检查后端服务是否运行：
   ```bash
   curl https://future-you-backend.loca.lt/api/health
   ```

3. 如果返回错误，重启服务：
   ```bash
   cd /home/ming/.openclaw/workspace-yuxin/future-you/server
   npm run dev
   ```

### 问题3：API 请求失败
**解决方法：**
前端会自动使用正确的后端地址。如果遇到 CORS 错误，清除浏览器缓存并刷新页面。
