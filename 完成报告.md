## 🚀 快速启动

### 前置条件
- Node.js >= 18.0.0
- MongoDB >= 4.4

### 启动步骤

1. **启动 MongoDB**
   ```bash
   sudo systemctl start mongod
   ```

2. **启动后端**
   ```bash
   cd server
   npm run dev
   ```

3. **启动前端**
   ```bash
   cd client
   npm run dev
   ```

4. **访问应用**
   ```
   http://localhost:5173
   ```

详细说明请参考 `启动指南.md`
